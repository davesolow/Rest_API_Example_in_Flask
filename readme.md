# Rest API Example Using Flask

This repository includes an example rest API using Python and Flask. The following web services are included:
1. GET Item
2. POST Item
3. GET Items
4. POST Authorization
5. POST Register
6. DEL Item
7. PUT Item
8. POST Store
9. GET Store
10. GET Stores
11. DEL Store

Other features:
1. JWT Encryption
2. SQL Alchemy
